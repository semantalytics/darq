DARQ

Example

Query for US state and US cities from two SPARQL endpoints. The example can be found in $DARQROOT/example/us/ .

Run it

  1. Download and setup DARQ
    1. get source code from svn
    2. set DARQROOT:  export DARQROOT=<path>
    3. [ set PATH:  export PATH=$PATH:$DARQROOT/bin ]
  2. go to $DARQROOT/example/us/
  3. run DARQ:
./darq --query q3.arq --config usDescr.n3 --verbose

Data

The data used for this example can be found at
    • http://www.daml.ri.cmu.edu/ont/USRegionState.daml
    • http://www.daml.ri.cmu.edu/ont/USCity.daml
(The files have been slightly modified.)
